# Park Tycoon

A browser-based park simulation game built with TypeScript, Vite, and AssemblyScript.

## GitHub Pages

Live site: https://verhaje.github.io/rollercoaster-vibecoon/

## Development

Install dependencies:

```bash
npm ci
```

Start local development server:

```bash
npm run dev
```

Build production assets:

```bash
npm run build
```

Run tests:

```bash
npm run test
```
